# sao-lx-express

> express

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao lx-express my-project
```

### From git

```bash
sao yoyoluxi/sao-lx-express my-project
```
